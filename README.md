# Welcome to Quiz Engine

### Stack Used:

- React 18
- Next.js 14.2.5

- npm version 10.8.2

### Running Steps:

1. `git clone https://github.com/lucasvribeiro/quiz-engine.git`
2. `cd quiz-engine`
3. `npm i` or `npm install`
4. `npm run dev`
